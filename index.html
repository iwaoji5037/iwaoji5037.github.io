# _Create a repository_ リポジトリを作成する。 「New repository」を押す。開いたページで「Repository name」にリポジトリ名を入力し「Create repository」を押すと作成される。 「Repository name」はGitHub上の個人のユーザー名、または組織名を着ける。 # _Clone the repository_ 「デスクトップに設定」と書いてある緑色のボタンを押す。GitHubのデスクトップアプリが開いたら、プロジェクトを保存する。 アプリが開かない場合は、アプリからリポジトリを複製しそれを起動する。 # _Create an index file_ インデックスファイルの作成。 好みのテキストエディタで、プロジェクトにindex.htmlファイルを追加する。 # _Commit & sync_ 同期・コミット。 追加、コミット及び編集をプッシュする。 # _… and you’re done!_ ### これでチュートリアルは終了だよ！ ブラウザを起動して(_http://ユーザ名.github.io._)へ
